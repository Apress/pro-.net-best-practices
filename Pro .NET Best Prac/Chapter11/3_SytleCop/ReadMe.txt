SAMPLE CODE: READ ME

1. To rebuild the project and run the StyleCop analysis, run the runner.bat command file.
2. The mandatory rule violations cause the build to fail.
3. To see all violations reported, open the Lender.Slos.Financial.StyleCop.xml file under the Artifacts\Analysis folder.
NOTE: There are more violations than those that broke the build.
4. Open the Lender.Slos.Express.sln solution with Visual Studio to resolve the mandatory violations.
5. Run the runner.bat command file and the build no longer fails.
